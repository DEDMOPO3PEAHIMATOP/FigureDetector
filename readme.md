### Программа генерации простейших геометрических фигур на однотонном фоне произвольного цвета.

**Размер фона:** 256х256.

***Фигуры:*** круг, треугольник, квадрат, ромб, гексагон

***Цвет фигур:*** произвольный, отличный от цвета фона

***Размер фигур:*** произвольный

***Положение на плоскости:*** произвольное

***Ограничения на построение фигур:***

Описывающие их прямоугольники должны иметь стороны размером не менее 25 пикселей и не более 150 пикселей
